À propos du Générateur d'UUID

Un outil simple pour tous vos besoins d'UUIDs

Qu'est-ce qu'un UUID?

Un UUID (Universally Unique Identifier) est un identifiant de 128 bits standardisé qui garantit l'unicité à travers l'espace et le temps. Les UUIDs sont largement utilisés en informatique pour identifier de manière unique des informations sans nécessiter de coordination centrale.

Les UUIDs sont représentés sous forme de chaînes de 36 caractères (32 caractères hexadécimaux et 4 tirets) comme ceci :

550e8400-e29b-11d4-a716-446655440000

Types d'UUIDs

  • UUID v1 - Basé sur l'heure et l'adresse MAC
  • UUID v4 - Générés aléatoirement
  • UUID v5 - Générés à partir d'un namespace et d'un nom
  • GUID - Format Microsoft Windows (identique à un UUID mais entouré d'accolades)

Fonctionnalités

  • Génération de différents types d'UUID (v1, v4, v5, GUID)
  • Génération en masse jusqu'à 1000 UUIDs
  • Création de QR codes à partir d'UUIDs
  • Génération de hashes (MD5, SHA-1, SHA-256, SHA-512)
  • Possibilité d'ajouter des préfixes et suffixes personnalisés
  • API REST gratuite pour intégration dans vos applications
  • Génération de snippets de code pour plusieurs langages
  • Interface en mode clair ou sombre

Utilisations courantes des UUIDs

  • Identifiants de base de données
  • Identifiants d'objets distribués
  • Références uniques dans les URLs
  • Tokens d'authentification
  • Identifiants de session
  • Noms de fichiers uniques
  • Identifiants d'instances dans les systèmes distribués
